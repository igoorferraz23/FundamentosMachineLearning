# Fundamentos de Machine Learning com Python

Este projeto contém vários notebook Jupyter voltado ao estudo de **fundamentos de machine learning**, com foco em conceitos fundamentais, diferentes tipos e aplicações práticas em Python.

## Tecnologias

- Python
- Jupyter Notebook
- pandas, numpy
- scipy
- plotly

## Execução

Instale as dependências e execute o notebook:

```bash
pip install -r requirements.txt
jupyter notebook
```

## Objetivo

Material educacional para estudo de estatística inferencial aplicada à análise de dados.
